# Data Analytics & Business Intelligence Portfolio 📊

Welcome to my Data Analytics portfolio! This repository contains a collection of SQL scripts and Power BI/Looker Studio dashboards demonstrating my ability to extract, clean, and visualize data for business decision-making.

## 📂 Projects Included:

### 1. Customer Support Operations Efficiency
*   **Goal:** Optimize workforce shift schedules by analyzing response times.
*   **Tools:** Advanced SQL (`TIMESTAMPDIFF`, `HOUR`), Microsoft Power BI.
*   **Files:** View the `Customer-Support-Operations` folder for the SQL extraction logic and the final dashboard visualization showing First Response Time (FRT) and Mean Time to Resolution (MTTR).

### 2. E-Commerce Customer Retention & RFM Analysis
*   **Goal:** Identify VIP customers and evaluate lifetime order values to improve retention strategies.
*   **Tools:** SQL, Microsoft Power BI.
*   **Files:** View the `E-Commerce-RFM-Analysis` folder for the relational database queries and the RFM dashboard.

### 3. Multi-Channel Marketing ROI Tracking
*   **Goal:** Track campaign success and optimize ad spend across multiple platforms.
*   **Tools:** Data Cleaning, Looker Studio.
*   **Files:** View the `Marketing-ROI-Tracking` folder to see the data aggregation methods and the visual ROI/CPC performance report.
